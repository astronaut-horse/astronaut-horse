The artist collaboration sessions linked below start with input images and text prompts selected by the artist. These are used to train a new stable diffusion concept which in turn produces several hundred output images. The artist then selects images from these sessions which are featured here.

The names of artists, input artworks and bulk output images are made public or private based on the artist's preferences.
