Open Call to Artists

This is an open call for visual artists to participate in an ongoing series of collaborations with a group of professional software engineers and machine learning researchers from the Recurse Center working to build open generative AI tools geared specifically towards empowering artists.

As of last month we have attained a hardware grant from a leading AI research and development group that allows us to host a live user-friendly version of our core application.

This application allows artists to train, prompt and generate images using ai models / concepts fine-tuned to their work.

As a visual artist and a human being I don't yet know how I feel about AI, especially generative tools that threaten to displace those of us lucky enough to be making a living through our work. This project is not a promotion for the use of AI tools in the arts, nor is it an attempt to capitalize on the current hype surrounding newly developed generative AI models.

This project is an open-ended experiment that seeks above all else to help artists attain critical agency in this rapidly evolving technical landscape.

This project is about...

  • ● building free tools
  • ● open experimentation
  • ● empowering artists
  • ● co-working over automation
  • ● building critical agency

Key points for artists...

  • ● your work remains yours
  • ● remain anonymous or be credited
  • ● train ai models using any images
  • ● freely prompt models to make images
  • ● use output images any way you want
  • ● optionally share images / feedback
  • ● optionally participate in exhibitions / events
  • ● use other tools as we build them
  • ● share ideas for new tools
  • ● share feedback on project

Check out the featured collaboration images and try out the current tools on our live application. If you want to collaborate further join our discord community or reach out to me directly.
