Astronaut horse refers to the ubiquitous image of an astronaut riding a horse that has come to act as a benchmark for the abilities of generative image models like Stable Diffusion.

Stable Diffusion is an open source generative image model whose code can be freely run, studied, manipulated and reimplemented. It is fundamentally distinct from previous generative image models like DALL-E and Midjourney which are for-profit ventures whose code and training details are not public.

This site hosts collections of output images from collaborations between visual artists (humans) and Stable Diffusion by way of by textual inversion training sessions.

The code and resources used for these collaborations is all available in the resources section of this site.
