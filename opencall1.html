Open Call to Artists

This is an invitation for visual artists to participate in an ongoing series of open-ended collaborations with a group of professional software engineers and machine learning researchers working to build freely available AI tools geared specifically towards empowering artists.

In February 2023 we applied for and received an ongoing hardware grant from a leading AI research and development group that allows us to host a live user-friendly version of our core application (linked below) and accommodate an increased number of collaborating artists.

As a visual artist and a human being I don't yet know how I feel about AI, especially generative tools that threaten to displace those of us lucky enough to be making a living through our work. This project is not a promotion for the use of AI tools in the arts, nor is it an attempt to capitalize on the current iteration of AI hype.

In short, this project is an open experiment that seeks above all else to help visual artists develop critical agency in this new technical landscape by inviting them to help create AI tools built specifically for artists and by artists.

The project is focused on...

  • ● building free AI tools for artists by artists
  • ● pursuing co-working over automation
  • ● building critical agency
  • ● empowering artists
  • ● avoiding hype

Key points for artists...

  • ● there are no obligations
  • ● sharing your artwork is optional
  • ● we respect the privacy of your data
  • ● AI-generated images can be used freely
  • ● optionally remain anonymous or be credited
  • ● optionally run custom trainings using your own images
  • ● freely prompt custom-trained models to make images
  • ● optionally share output images with community
  • ● optionally participate in exhibitions / events
  • ● learn practical coding skills if you want
  • ● use new tools as we build them
  • ● share your thoughts

If you're curious about potentially collaborating check out the featured collaboration images linked below and play around with the current tools on our live application.

If you're interested in collaborating further with custom trainings, new tools, discussions, exhibitions etc. check out our discord community, we'd be exited to work with you.
